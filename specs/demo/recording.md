# AgriTech Platform - Demo Video Script

**Target Duration:** 8-10 minutes  
**Format:** Screen recording with voiceover  
**Audience:** Potential customers, investors, partners

---

- [ ] Clear browser cache and use fresh test account
- [ ] Prepare sample data (farms, parcels, tasks, invoices)
- [ ] Test all features before recording
- [ ] Close unnecessary browser tabs/applications
- [ ] Use high-quality microphone

## Post-Production Notes

### Editing Tips
1. **Add text overlays** for key features mentioned
2. **Zoom in** on important UI elements
3. **Add transitions** between major sections
4. **Include background music** (subtle, professional)
5. **Add captions/subtitles** for accessibility
6. **Create chapter markers** for easy navigation
7. **Add call-to-action buttons** at key moments
8. **Include logo watermark** (subtle, corner placement)

### Technical Specifications
- **Resolution:** 1920x1080 (Full HD) minimum
- **Frame Rate:** 30fps or 60fps
- **Audio:** 48kHz, stereo
- **Format:** MP4 (H.264)
- **Aspect Ratio:** 16:9
- **Duration:** 8-10 minutes (ideal for demo videos)

### Distribution
- Upload to YouTube (unlisted or public)
- Embed on website landing page
- Share on social media
- Include in email campaigns
- Add to investor pitch deck

## Recording Tips

1. **Practice first** - Do a dry run without recording
2. **Speak clearly** - Enunciate and maintain steady pace
3. **Use cursor highlights** - Make cursor visible and smooth
4. **Avoid mistakes** - If you make an error, pause and re-record that section
5. **Show real data** - Use realistic sample data, not placeholder text
6. **Highlight key features** - Pause briefly on important UI elements
7. **Keep it natural** - Don't rush, but maintain energy
8. **Test audio** - Record a test clip first to check audio levels
