## 5. STYLE VISUEL PAR PLATEFORME

### LinkedIn
- Format : 1200x627px (ratio 1.91:1) ou 1080x1080 pour carrousels
- Style : Professionnel, épuré, beaucoup d'espace blanc
- Photos : Vraies photos de terrain agricole marocain — ❌ pas de stock générique
- Texte sur image : Minimal, 5-7 mots maximum
- Carrousels : Fond anthracite (#1E1E1E), texte blanc, une idée par slide

### Instagram
- Reels : Format 1080x1920 (9:16), sous-titres TOUJOURS présents
- Carrousels : 1080x1350 (4:5) pour plus de visibilité dans le feed
- Style : Dynamique, couleurs vives, rythme rapide
- Première slide : Accroche visuelle forte — question ou chiffre choc
- Dernière slide : CTA clair + logo AGROGINA

### Facebook
- Reprendre le contenu Instagram adapté
- Vidéos : Peuvent être plus longues (jusqu'à 60 secondes)
- Ton : Légèrement plus accessible pour toucher les agriculteurs 40+

## 6. RÈGLES DE COMPOSITION VISUELS

### Photos de terrain
- Privilégier paysages agricoles marocains (Souss, Gharb, Saïss...)
- Golden hour (lever/coucher de soleil) pour la chaleur
- Overlay sombre 40-60% d'opacité si texte par-dessus

### Aperçus de l'application — RÈGLE DE PROTECTION
- **TOUJOURS floutés ou partiels**
- Montrer 2-3 secondes maximum dans les vidéos
- Angle : légère perspective 3D — ❌ jamais de screenshot plat
- ❌ Ne jamais montrer l'architecture complète des modules

## 7. TON DE COMMUNICATION

### Ce qu'on est
- Direct et concret
- Accessible (pas de jargon technique inutile)
- Confiant sans être arrogant
- Proche du terrain agricole marocain
- Bilingue naturel (français / arabe)

### Ce qu'on n'est PAS
- Scientifique et austère
- Complexe et technique
- Institutionnel et froid
- Générique (on parle toujours du Maroc, de l'agriculture marocaine)

### Exemples de ton

| ❌ À éviter | ✅ À utiliser |
|---|---|
| "Notre solution d'optimisation des ressources agricoles..." | "Fini les appels entre votre agronome et votre comptable." |
| "Algorithme de machine learning appliqué à l'agronomie" | "L'IA qui connaît vos stocks avant de vous conseiller" |
| "Nous sommes leaders de l'AgriTech marocaine" | "Votre ferme mérite un vrai copilote digital" |
| "Solution scalable et interopérable" | "Tout dans une seule application" |

## 8. RÈGLE D'OR — MONTRER SANS RÉVÉLER

AGROGINA est nouveau sur le marché. La stratégie est de créer le désir
sans donner le mode d'emploi aux concurrents.

| ✅ CE QU'ON FAIT | ❌ CE QU'ON ÉVITE |
|---|---|
| "Gérez tout depuis une seule interface" | Montrer l'architecture des modules |
| Aperçu flouté du dashboard (2-3 secondes) | Démo complète d'une fonctionnalité |
| "L'IA connecte vos décisions" | Expliquer l'algorithme d'AgromindIA |
| "15-25% d'économies observées" | Détailler sur quoi et comment |
| "Venez voir au SIAM" | Tout montrer en ligne |

## 9. STRATÉGIE ÉDITORIALE

### Répartition du contenu

| Type | % | Description |
|---|---|---|
| Éducatif / Valeur | 45% | Problèmes du secteur, tendances, conseils. TOUJOURS conclure par "AGROGINA résout ça" sans détailler comment |
| Produit / Aperçus | 20% | Montrer que ça existe : écrans floutés, fonctionnalités évoquées sans détail |
| Problème / Solution | 15% | Douleurs concrètes agriculteurs → "La solution existe, venez voir au SIAM" |
| Vision / Marque | 10% | Pourquoi AGROGINA existe, convictions, l'avenir de l'agriculture |
| SIAM / CTA | 10% | Teasing salon, formulaire RDV, countdown (augmente en phase 3) |

### Règles LinkedIn
- Longueur optimale : 1300-2000 caractères
- Structure : Accroche forte → Développement → CTA
- Fréquence : 4 posts/semaine (mardi, mercredi, jeudi, vendredi)
- Horaires : 8h-9h ou 12h-14h (heure Maroc)
- Hashtags : 3-5 maximum, en fin de post
- ⚠️ Lien : JAMAIS dans le post — toujours en commentaire
- Engagement : Répondre aux commentaires dans l'heure

### Règles Instagram
- Reels : 7-15 secondes idéal, 30 secondes maximum
- Caption : Court et punchy, 100-150 caractères + hashtags
- Fréquence : 5-6 Reels + 2-3 carrousels par semaine
- Hashtags : 8-12, mix populaires et niches
- Stories : Quotidiennes, sondages et questions pour l'engagement
- Hook : Les premières secondes doivent stopper le scroll

## 10. PLANNING SIAM — 3 PHASES

### Phase 1 — FONDATION (Semaines 1-3 : 2-22 Mars)
**Objectif : Établir la présence, montrer l'expertise, créer la curiosité**
- 60% Éducatif/Valeur
- 25% Produit/Aperçus
- 15% Vision/Marque
- 0% SIAM (trop tôt)

### Phase 2 — CRÉDIBILITÉ (Semaines 4-6 : 23 Mars - 12 Avril)
**Objectif : Montrer la solution, prouver la valeur, introduire SIAM**
- 40% Éducatif/Valeur
- 25% Produit/Aperçus
- 20% Problème/Solution
- 15% SIAM/CTA

### Phase 3 — CONVERSION (Semaines 7-8 : 13-26 Avril)
**Objectif : Remplir l'agenda de RDV, créer l'urgence**
- 25% Éducatif/Valeur
- 15% Produit/Aperçus
- 15% Problème/Solution
- 10% Vision/Marque
- 35% SIAM/CTA ← Maximum d'urgence ici

## 11. GUIDE PROMPTS VISUELS

### Structure prompt Whisk
```
[TYPE] + [SUJET] + [COMPOSITION] + [ÉCLAIRAGE] + [COULEURS] + [STYLE] + [FORMAT] + [EXCLUSIONS]
```

**Exemple :**
```
Professional photograph, aerial view of Moroccan agricultural fields
with modern irrigation system, rule of thirds composition with farmhouse
in lower left, golden hour warm sunlight from the right, dominant forest
green fields with blue water channels, photorealistic style with slight
color grading, 16:9 ratio landscape format, no people, no text, no watermarks
```

### Vocabulaire Whisk clé
```
Style photo    : photorealistic, professional photograph, editorial style
Style graphique: flat vector design, minimalist illustration, clean geometric
Éclairage      : golden hour, soft natural light, bright daylight
Composition    : rule of thirds, centered, aerial view, eye level, wide shot
Ambiance       : professional, modern, clean, warm, trustworthy, innovative
Qualité        : high resolution, sharp focus, detailed, crisp, 4K quality
Exclusions     : no text, no watermarks, no people faces, no logos
```

### Structure prompt Veo (vidéos 8 secondes)
```
[STYLE VISUEL] + [ACTION/MOUVEMENT] + [SUJET] + [MOUVEMENT CAMÉRA] + [ÉCLAIRAGE] + [AMBIANCE] + [RYTHME]
```

**Exemple :**
```
Modern corporate video style, 2026 aesthetic, hands typing on tablet
showing agricultural dashboard with green and blue interface colors,
slow push-in camera movement toward the screen, bright natural office
lighting from large windows, professional and innovative mood,
smooth steady rhythm, clean crisp digital quality not filmic grain
```

### Vocabulaire Veo clé
```
Style global   : modern 2026 aesthetic, contemporary corporate video, tech startup energy
Qualité        : crisp digital quality, clean sharp footage, NOT filmic grain, NOT vintage
Mouvement cam  : slow push-in, smooth tracking shot, gentle pan, subtle zoom
Éclairage      : bright natural daylight, soft diffused light, golden hour warmth
Rythme         : smooth steady pace, dynamic but controlled, calm professional rhythm
À ÉVITER       : filmic grain, vintage look, sepia tones, dark moody, shaky camera
```

### Transitions entre segments Veo
Pour vidéos > 8 secondes, terminer chaque segment par :
- "...ending with camera movement toward [élément début segment suivant]"
- "...smooth transition preparing for next scene showing [description]"
