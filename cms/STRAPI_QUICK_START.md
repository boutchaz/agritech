## Starting Strapi

### 1. Start the Services

```bash
cd /Users/boutchaz/Documents/CodeLovers/agritech
docker-compose up -d strapi strapi-db
```

### 2. Monitor Initialization

```bash
docker-compose logs -f strapi
```

Wait for the message: `Server started on port 1337`

### 3. Access Admin Panel

Open in your browser:
- **Local**: http://localhost:1337/admin
- **Traefik**: http://cms-agritech-dashboard-g6sumg-2b12b9-5-75-154-125.traefik.me/admin

### 4. Create Admin User

On first access, you'll be prompted to create an admin user:
- **First Name**: Your name
- **Last Name**: Your last name
- **Email**: Your email
- **Password**: Strong password (min 8 characters)

### Useful Commands

```bash
# View logs
docker-compose logs -f strapi

# Restart Strapi
docker-compose restart strapi

# Stop Strapi
docker-compose stop strapi

# Access Strapi container shell
docker-compose exec strapi sh

# View database
docker-compose exec strapi-db psql -U strapi -d agritech_strapi
```

## Troubleshooting

### Port Already in Use
If port 1337 is already in use, edit `docker-compose.yml`:
```yaml
ports:
  - "1338:1337"  # Change 1337 to 1338
```

### Database Connection Issues
Check if strapi-db is running:
```bash
docker-compose ps strapi-db
```

### Reset Strapi (Fresh Start)
```bash
docker-compose down
docker volume rm agritech_strapi-db-data
rm -rf strapi/app
docker-compose up -d strapi strapi-db
```

```bash

STRAPI_ADMIN_JWT_SECRET=<auto-generated>
STRAPI_APP_KEYS=<auto-generated>

STRAPI_URL=http://localhost:1337

VITE_STRAPI_API_URL=http://localhost:1337/api
```
